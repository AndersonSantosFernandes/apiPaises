Desafio obrigatório do curso de desenvlovimento web na plataforma CoderHouse

O desafio consiste em criar um sistema simple para consumo de um API pública, onde o site escolhido foi o do IBGE que fornece dados de vários paíse de todos os continentes.

Através do select é possivel selecionar um país qualque e a logica junto com a API trás as informações de cada um.

O interessante dasta API é que tem a opção de exibir informações históricas de cada país; para quem gosta de história pode ser uma boa fonte de informação.

A interface possui responsividade
Sistema criado por Anderson Santos Fernandes atrvés de conhecimentos adquiridos no curso de dev web da CODERHOUSE.
